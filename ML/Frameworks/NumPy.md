* Создание `x = np.array([...])
* Аттрибуты
	* `x.ndim` - размерность
	* `x.shape` - форма
	* `x.dtype` - тип данных
* Расширение - при поэлементных операциях с тензорами разной размерности (расширение виртуально, фактически не происходит):
	* добавляются оси расширения (до совпадения формы)
	* меньший тензор копируется для заполнения расширенного
	* проводятся поэлементные операции
	* (32,10) + (10,) -- (1,10) -- (32,10)
* Мат операции
	* Скалярное произведение `np.dot(x,y)
	* Изменение формы `x = x.reshape((6, 1))
	* Транспонирования `x = np.transpose(x)
